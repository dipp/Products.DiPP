Publikation wissenschaftlicher Artikel
======================================

Das Publizieren von wissenschaftlichen Artikel unterscheidet sich erheblich vom
Erstellen einfacher :term:`HTML` Seiten, die mit :term:`Plone` eigenen Bordmitteln
direkt im Browser erstellt werden.

wissenschaftliche Artikel werden im Gegensatz dazu mit einer Textverarbeitung
erstellt, im :term:`RTF`-Format abgespeichert. Auf dem Server wird das Document
dann über das :term:`DocBook` XML Format zu :term:`HTML` konvertiert. 

Einstiegspunkt für das Konvertieren der Artikel ist immer die Editoral Toolbox,
die über die sogenannte :term:`Personal bar` am oberen Rand oder in der linken
Navigation erreichbar ist.
